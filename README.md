# 8AE Portfolio

A dependency-free personal portfolio for GitHub Pages.

## Publish with GitHub Pages

1. Create a GitHub repository named `8AE.github.io` (for `https://8ae.github.io`) or any repository you prefer.
2. Upload these files to its default branch.
3. In the repository, open **Settings → Pages**, choose **Deploy from a branch**, then select the default branch and the `/ (root)` folder.
4. Save. GitHub will publish the site shortly after.

The Pokémon game link is already configured as `https://8ae.github.io/whos_that_pokemon/`.
